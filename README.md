Student Management CRUD Application
A basic Spring Boot application for managing students, featuring complete CRUD operations. The application uses JPA repository with PostgreSQL for data persistence, and Data Transfer Objects (DTOs) for request validation and response handling, ensuring robust data management and integrity.

Features:

Create, Read, Update, and Delete (CRUD) student records.
JPA repository integrated with PostgreSQL database.
DTO-based request validation.
Response DTOs for successful operations.
Comprehensive validation for resource management.
